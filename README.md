## Deploying on GitHub Pages

This repo is still deployable as a static GitHub Pages site. Configure Pages to serve from the repository root and publish `index.html`.

Local preview:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173
```
